# Asset provenance

The directories `Tiny Swords (Free Pack)`, `Tiny Swords (Update 010)` and
`Tiny Swords (Enemy Pack)` contain files originating from Pixel Frog's Tiny Swords packs obtained
separately from this source repository. `index.json` is a generated file inventory and
`lindocara-asset-catalog.json` is LindoCara's semantic catalogue over those files.

This note records provenance only; it does not state, replace or extend the packs' licence terms.
Before redistributing these source files or distributing a build that contains them, verify the
terms attached to the original purchase or download with the relevant rights holder.
